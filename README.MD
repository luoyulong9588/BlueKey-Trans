# ⌨️ 蓝牙键传 (BlueKey-Trans) —— 物理隔离内网文件传输神器

**【项目背景】**
在很多高安全级别的内网环境下，电脑的 USB 接口被禁用（无法识别 U 盘、移动硬盘），且没有外网连接。但是，系统通常允许插入 **USB 键盘**。

**蓝牙键传** 利用 ESP32-S3 模拟一个合法的“物理键盘”，通过手机蓝牙将文件或文字发送给硬件，再由硬件以“疯狂敲键盘”的方式，把数据输入到电脑中。从而完美绕过内网对 USB 存储设备的限制。

### 🚀 核心卖点
* **免驱动、免安装**: 电脑识别它就是个普通键盘，不需要在电脑上装任何软件。
* **双模传输**: 
    - **文字模式**: 粘贴长篇报告、代码、中文、特殊符号。
    - **文件模式**: 传输 Word、Excel、图片等任意文件（建议 100KB 以内效果最佳）。
* **极速压缩**: 采用 Gzip 压缩，本来要敲 10 分钟的内容，现在 3 分钟搞定。
* **智能防乱码**: 自动模拟按下 Shift 键，强制让电脑切换到英文模式，防止输入法干扰。
* **断网可用**: 手机与硬件通过蓝牙连接，硬件与电脑通过 USB 连接，全程不需要互联网。

---

### 🛠️ 怎么用？

1.  **准备硬件**: 将 `S3_Firmware.ino` 烧录到 ESP32-S3 开发板，插到内网电脑上。
2.  **打开接收页**: 在电脑浏览器打开 `receiver.html`。
3.  **手机操作**: 
    - 手机浏览器访问 `sender.html`。
    - 蓝牙连接到“S3-Keyboard-Link”。
    - 粘贴一段话，或者选个文件，点“启动物理注入”。
4.  **坐等收货**: 你会看到电脑开始自动打字，打完后，接收页会自动还原出你的文件或中文。

---

### ⚠️ 注意事项
* **焦点不能丢**: 传输时，电脑上的接收网页必须在最前面，别去点别的窗口。
* **输入法**: 虽然硬件会尝试自动切英文，但手动把电脑输入法关掉或切到 ENG 模式最稳。
* **大小限制**: 因为是模拟打字，传大文件会比较慢（适合传文档、密钥、报表）。
